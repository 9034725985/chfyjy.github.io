Solar System Scope Texture Library

Solar System Scope provides high quality, free to use textures for objects in the solar system.

Used in Qt 3D planets-qml example.

The sources can be found in tqtc-qt5/qt3d/examples/qt3d/planets-qml/images/solarsystemscope.
